两个链表的第一个公共节点

Carpe diem
2018-05-13
Coding Interviews, 链表

问题描述

输入两个链表，找出它们的第一个公共结点。

思路

  1. O(mn)的复杂度,遍历两次
  2. 利用单链表的隐藏特性,两个拥有公共节点的单链表形状是一个Y型,则从其相交之后则相同,则我们可以思考从最后往前进行比较,找到第一个不同的。则后进先出。用栈实现,或者先得到其长度，倒数相同的索引开始向前遍历,找到第一个相同的。但是求长度也需要额外的时间复杂度呀。
  3. 倒置单链表,再从头开始。
